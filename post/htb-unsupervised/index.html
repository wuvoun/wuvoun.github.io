HTB-Unsupervised

  • 题目简介

    The incident happened around 4:30 PM on Friday, “The Last day of the week” at the Accounts/Marketing department of “Finance XYZ” company. There weren’t many people in the department, and the remaining were not paying much attention to their surroundings as they were getting ready to head home. After the weekend on Monday while reviewing the security cam footage member of an IT team saw “Eddie” a new intern in the Accounts/Marketing department, plugging a USB into an unauthorized computer (containing sensitive financial and marketing documents), interacting with computer and unplugging the USB before heading out. As the incident happened 2 days ago, and not having enough knowledge of what Eddie did the security team use caution while asking around and gathering intel to avoid causing suspicion. The only information they were able to find out was that Eddie had a “Toshiba” USB. You are provided with a partial image of the “unauthorized computer" as well as a list of important documents, to investigate what he did and if he stole something sensitive or not?

AD1文件查看

将 Image.ad1 文件导入 AccessDate FTK Imager

File → Add Evidence Item → Image File

SYSTEAM 文件分析

将 SYSTEAM 文件导出，使用 Registry Explorer 打开

常见取证信息：

  • SYSTEM\MountedDevices → 已挂载设备痕迹
  • SYSTEM\CurrentControlSet\Enum\USBSTOR → USB 历史
  • SYSTEM\CurrentControlSet\Services → 系统服务 / 驱动 (可发现异常驱动、持久化机制)
  • SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\Interfaces → 网络相关
  • SYSTEM\CurrentControlSet\Control\TimeZoneInformation → 时区信息
  • SYSTEM\Select → 控制集选择信息(指明系统当前使用的是哪个 ControlSet)
  • SYSTEM\CurrentControlSet\Control\ComputerName\ActiveComputerName → 主机名
  • SYSTEM\CurrentControlSet\Control\CrashControl → 蓝屏/内存转储配置

SOFTWARE 文件分析

常见取证信息：

  • SOFTWARE\Microsoft\Windows NT\CurrentVersion → 系统版本、安装时间、注册用户信息
  • SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall → 已安装软件清单（名称、版本、安装时间）
  • SOFTWARE\Microsoft\Windows\CurrentVersion\Run → 开机自启项（常见持久化位置）
  • SOFTWARE\Microsoft\Office → Office 版本及配置信息
  • SOFTWARE\Adobe → Acrobat / Flash 等 Adobe 软件信息
  • SOFTWARE\Microsoft\Windows\CurrentVersion\Internet Settings → IE 浏览器及代理设置
  • SOFTWARE\Wow6432Node → 32 位应用安装信息（在 64 位系统上）
  • SOFTWARE\JavaSoft → Java 运行环境信息
  • SOFTWARE\Microsoft.NETFramework → .NET 运行环境信息
  • SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\LogonUI ->可以定位到最近交互式登录的账户

ink文件分析

C:\Users\MrManj\AppData\Roaming\Microsoft\Windows\Recent\ 文件夹是 Windows 系统用来记录用户最近打开的文件和文件夹的快捷方式 的地方，每当用户在 Windows 中打开一个文件或文件夹时，系统会在这个目录下生成一个对应的 .LNK 文件（快捷方式）。

这个文件夹就像一个日志，记录了用户最近打开过哪些文件和文件夹，但不会存文件内容本身。

用 LECmd.exe 导出内容为csv

LECmd.exe -d ./ --all --csv .

UsrClass.dat 文件分析

从AccessDate FTK Imager 中转储出 UsrClass.dat , 其中可以恢复用户浏览过的文件夹，重建访问路径，辅助判断文件来源

将UsrClass.dat 导入 ShellBags Explorer

Desktop → E:\ → FTK Imager / Documents / Important docs / Business Proposals 表示 用户在桌面下，访问过 E 盘的 Documents 文件夹，以及其子目录 Important docs 和 Business Proposals

同时，用户在F:\ 也有一个Documents 文件夹，可能是 U 盘

1. Find out the time zone of victim PC. (UTC+xx:xx)

UTC 时间 − (-300 分钟) = UTC 时间 + 300 分钟 = UTC 时间 + 5 小时

所以，ActiveTimeBias = -300 就代表时区 UTC+05:00。

2. Employees should be trained not to leave their accounts unlocked. What is the username of the logged in user?

全局搜索 LogonUI ,定位到最近交互式登录的账户，发现用户 MrManj

3. How many USB storage devices were attached to this host in total?

定位 SYSTEM\CurrentControlSet\Enum\USBSTOR ,查看有多少种 Desk Id

发现三种usb设备

4. What is the attach timestamp for the USB in UTC?

根据题目可知，usb是一个 Toshiba usb设备，找到设备为 TOSHIBA TransMemory USB Device ,USB 的连接时间戳（UTC）是 2024-02-23 11:37:50

5. What is the detach timestamp for the USB in UTC?

Last Removed 这个字符串种有最后分离时间戳 2024-02-23 11:39:12

6. Which folder did he copy to the USB?

从对 UsrClass.dat 的文件分析，可以知道复制了 Documents 目录

7. There were subfolders in the folder that was copied. What is the name of the first subfolder? (Alphabetically)

按字母排，文件夹为 Business Proposals

8. Eddie opens some files after copying them to the USB. What is the name of the file with the .xlsx extension Eddie opens?

由分析的ink文件，发现打开的文件为 Business Leads.xlsx

9. Eddie opens some files after copying them to the USB. What is the name of the file with the .docx extension Eddie opens?

Proposal Brnrdr ltd.docx 是Eddie 打开的扩展名为 .docx 的文件

10. What was the volume name of the USB?

查找HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Portable Devices\Devices\<DeviceID> ，发现卷名为 RVT-9J

11. What was the drive letter of the USB?

由上题可知USB的盘符为 E

12. I hope we can find some more evidence to tie this all together. What is Eddie’s last name?

C:\Users\<用户名>\AppData\Local\Microsoft\Windows\Explorer\ 存在像素图

使用 thumbcache viewer 打开像素图,知道名为 Homer

13. There was an unbranded USB in the USB list, can you identify it’s manufacturer’s name?

查看注册表 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USB ，寻找到usb的供应商 ID (VID) 和产品 ID (PID)： VID_346D&PID_5678

Ven_XXX = 厂商，Prod_XXX = 产品型号

google搜索查到制造商为 Shenzhen SanDiYiXin Electronic Co.,LTD
