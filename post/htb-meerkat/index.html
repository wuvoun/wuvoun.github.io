签名 (Signature) 含义 风险等级 分析
ET CINS Active Threat Intelligence Poor Reputation IP group xx 来自 CINS 情报源的恶意/低信誉 IP 中 外部恶意节点扫描或探测，通常是背景噪声。建议封禁或加入黑名单。
ET DROP Dshield Block Listed Source group 1 来自 DShield 黑名单的威胁 IP 中 全球攻击源的 IP，被监测到访问你的主机。建议在防火墙层面阻断。
ET POLICY GNU/Linux APT User-Agent Outbound likely related to package management Linux 主机使用 APT 下载更新包 低 若是服务器正常更新可忽略；如出现在非预期机器，需检查是否被用作下载恶意 payload。
ET EXPLOIT Bonitasoft Authorization Bypass (CVE-2022-25237) Bonitasoft BPM 平台存在认证绕过漏洞 高 攻击者可利用绕过认证，进一步上传文件或执行命令。需立即检查系统是否运行 Bonitasoft，并升级补丁。
ET EXPLOIT Bonitasoft Authorization Bypass and RCE Upload (CVE-2022-25237) 针对 Bonitasoft 的 RCE 漏洞利用 高 表明攻击者可能上传并执行恶意代码。必须立刻检测是否有 Webshell 或异常进程。
ET EXPLOIT Bonitasoft Successful Default User Login Attempt 攻击者使用 Bonitasoft 默认账户尝试登录 高 表示弱口令风险，攻击者可能进入后台。应立即禁用默认账户并审计日志。
ET WEB_SPECIFIC_APPS Bonitasoft Default User Login Attempt 同上，弱口令或默认账户攻击迹象 高 检查 Bonitasoft 平台安全配置。
ET INFO User-Agent (python-requests) Inbound to Webserver 外部请求使用 Python requests 库发起 中 常见于扫描脚本或攻击工具（PoC/自动化漏洞利用）。需结合上下文确认是否恶意。
GPL WEB_SERVER DELETE attempt 可能的 Web 攻击尝试（DELETE 方法） 中 一般用于探测或删除资源。需确认目标 Web 应用是否正确处理 HTTP 方法。
ET ATTACK_RESPONSE Possible /etc/passwd via HTTP (linux style) 响应中可能包含 /etc/passwd 文件内容 高 表明存在 LFI/RFI 或命令执行漏洞已被利用。必须立即检查是否有数据泄露。
null 无 signature 字段，可能是解析异常或无效告警 低 建议检查 Suricata 规则库和日志完整性。
