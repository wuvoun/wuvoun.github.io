  1. It is crucial to understand any payloads executed on the system for initial access. Analyzing registry hive for user happy grunwald. What is the full command that was run to download and execute the stager.

NTUSER.DAT 是 Windows 用户注册表配置单元 (registry hive) 文件，它保存了用户级别的配置和活动痕迹。每个用户登录系统时，都会有一个 NTUSER.DAT 文件存放在该用户的 用户配置目录 下，它其实就是注册表的 HKEY_CURRENT_USER (HKCU) 分支，里面包含：

  • 软件配置：应用的设置、偏好。
  • 最近使用记录：比如运行过的命令、访问过的路径。
  • 持久化位置：启动项

NTUSER.DAT 文件常见取证路径：

  • HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU

    → 用户在运行框里输入过的命令。

  • HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\UserAssist

    → Base64/ROT13 编码的执行过的程序路径。

  • HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\TypedPaths

    → 资源管理器输入过的路径。

  • RecentApps、ComDlg32

    → 最近使用的应用/文件
