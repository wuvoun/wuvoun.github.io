HTB-DroidPhish

dd 文件通常指的是用 dd 命令生成的文件，通常是磁盘镜像，引导扇区备份，内存转储文件 ，二进制数据块等

 file DroidPhish.dd
DroidPhish.dd: Linux rev 1.0 ext4 filesystem data, UUID=e98cc545-b7fe-4ba9-8b33-2fb9bba476d6, volume name "Android-x86" (needs journal recovery) (extents) (large files)

通过file指令，可以说明 DroidPhish.dd 文件里是一个 ext4 文件系统镜像，通过DiskGenius打开

主要目录：

  • /data/data/<package> → 表示该应用在设备上确实安装并被使用
  • /data/app → 用户安装的 APK
  • /data/system → 系统配置、账户、锁屏信息
  • /data/misc → 网络配置、keystore、logd 缓存
